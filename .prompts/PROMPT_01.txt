`sources.yaml` (authoritative source registry) minimal schema:
- top-level `sources:` list of source entries (canonical shape)
- per source required fields:
  - source_id (stable slug, filesystem-safe)
  - display_name
  - enabled (bool)
  - crawlability (PublicHtml | Api | Rss | Gated | ManualOnly)
  - mode (crawler | manual)
  - listing_urls (list; empty allowed for manual-only sources)
- per source optional fields:
  - detail_url_patterns
  - rate_limit (requests/min or equivalent config object)
  - notes

All later prompts that read `sources.yaml` must use this schema consistently (including the top-level `sources:` key).
